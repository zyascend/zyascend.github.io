简析React Hooks

是什么

简单来说，React Hooks可以帮助函数式组件获得一般组件的功能，如

  • 不需要this使用state
  • 使用生命周期函数

useState

在函数组件中使用state

useState

作用

在函数组件中使用生命周期函数

用法

1.模拟 componentDidMount

2.模拟willUpdate 每次渲染时调用

3.模拟willUnMount 在卸载时执行

注意点

  • 多个useEffect要写在一起，中间不能被打断
  • 不能用判断逻辑包裹useEffect

useRef

作用

  • 如同class组件中的ref 便于操作dom
  • 修改被useEffect包裹的state
  • 作为一个全局作用域，可以被所有地方使用，同时统一更新

使用

1.作为全局变量

2.用于操作dom

useMemo

缓存对象 根据特定的值是否发生改变而决定是否更新整个对象

useCallback

同上 缓存方法 防止由于重新生成相同方法而造成不必要的渲染
