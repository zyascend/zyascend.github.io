provide & inject

https://cn.vuejs.org/v2/api/#provide-inject

这对选项需要一起使用，以允许一个祖先组件向其所有子孙后代注入一个依赖，不论组件层次有多深，并在起上下游关系成立的时间里始终生效。

使用

父组件提供一个可被子组件读取的provide

Javascript

不足

  1. inject之后，provide的数据不是响应式的 this.name 无法监听parent中的变动
  2. inject某个需要的变量之后 此变量不可再次申明 this.name已被占用
  3. 仅支持Vue.js 2.2.0 以上版本
